功能描述 V1.0

BI门户在功能上与IE收藏夹类似，都是用来收集各种资源地址，但是也有明显的不同和限定：

  1. 主要收集来自BI、报表或者其他管理系统的链接资源
  2. 由管理员收集整理，统一编排，而不是由用户自己操作
  3. 这些资源可以分配给不同的角色，而用户可以拥有多个角色
  4. 该系统可以单独使用，也可以与CAS系统集成，用户对某一资源的访问权限最终由目标系统决定
  5. 内连接网页高度，可以在chrome控制台通过document.body.scrollHeight获得

V1.0 明确不实现以下功能

  1. 数据权限
  2. 向目标系统动态传递参数
